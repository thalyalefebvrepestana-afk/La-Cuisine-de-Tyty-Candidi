# La-Cuisine-de-Tyty-Candidi

Petit site de démonstration « La Cuisine de Tyty » — collection de recettes simples.

Démarrage rapide

1. Ouvrir un terminal dans le dossier du projet.
2. Lancer un serveur local (Python) :

```bash
python3 -m http.server 8000
```

3. Ouvrir http://localhost:8000 dans votre navigateur.

Fichiers ajoutés

- `index.html`
- `styles.css`
- `scripts.js`

Notes

Ce site est une création originale inspirée par des sites de recettes — il ne reproduit pas de contenu protégé. Vous pouvez étoffer les recettes, ajouter un système de pages individuelles, ou intégrer un CMS si besoin.
